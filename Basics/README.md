# 📖 Welcome to the Basics Section

Welcome to the **Basics** section of this repository!

---

## 🎯 Purpose

In this section, you will learn the fundamentals needed to understand web security, HTTP behavior, and common application logic patterns.

---

## 📚 What you will find here

This section includes introductory exercises covering topics such as:

- HTTP status codes and their meanings  
- Headers manipulations
- HTTP Methods
- Usage of the library `requests` 
  

---

## 🚀 Goal

The goal of this section is to build a strong foundation before moving on to more advanced security challenges and CTF-style exercises.

---

## 🧠 Recommendation

Take your time with each exercise and make sure you understand **why** something works, not just how.
